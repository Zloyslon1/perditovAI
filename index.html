Урок

Ты соберёшь своего первого AI-ассистента в Telegram через Factory Installer. Запустишь установщик одной командой и получишь живого бота.

Подготовка и детали

Перед стартом подготовь 4 вещи:

  1. Терминал — на Mac: iTerm2
  2. Telegram-бот — через @BotFather
  3. Telegram User ID — через @userinfobot
  4. API key — OpenRouter или OpenCode / Zen

Команда установки:

bash <(curl -fsSL https://raw.githubusercontent.com/tonytrue92-beep/openclaw-factory/main/scripts/demo-install.sh)
⚠️ bot token и API key никому не отправляй. Username бота и token — это не одно и то же.

VIP — 5 AI-сотрудников в Telegram
VIP

VIP-тариф включает команду из 5 агентов, каждый в своём Telegram-боте:

  • 🔧 Технарь — ошибки, интеграции, настройки
  • 📈 Маркетолог — смыслы, контент, воронки
  • 🎬 Продюсер — продукт, сценарии, запуски
  • 🎨 Дизайнер — визуалы, баннеры, креативы
  • 🧭 Координатор — задачи и управление
Инструкция по установке

⚠️ Что нужно до старта:

  1. Установленный OpenClaw из 1 недели
  2. 5 Telegram-ботов через @BotFather
  3. Telegram ID через @userinfobot

Шаг 1. Получите VIP-токен

  1. Откройте @AITeamVIPBot
  2. Нажмите /start
  3. Введите email или телефон от оплаты
  4. Скопируйте VIP-токен

Шаг 2. Запустите установщик

bash <(curl -fsSL https://raw.githubusercontent.com/tonytrue92-beep/openclaw-agents-pack/main/scripts/install-agents.sh)

Шаг 3. Выберите 2) VIP — 5 агентов

Шаг 4. Проверьте — напишите любому боту в Telegram.

🆘 Если что-то не получилось:

  • Токен не прошёл → скопируйте заново из @AITeamVIPBot
  • Бот не нашёл вас → проверьте email/телефон, подождите 30 мин
  • OpenClaw не найден → сначала установите из 1 недели
  • Ничего не помогло → соберите debug-архив:
bash <(curl -fsSL https://raw.githubusercontent.com/.../install-agents.sh) --collect-debug
Уже стоят 3 агента со Стандарта? Запусти установщик снова → выбери VIP → «Дополнить». Старые агенты останутся с памятью, добавятся Дизайнер и Координатор.
